## ✨ 6 Learning Modules

### 🔤 Letters (A–Z)
- All 26 letters with emoji, word, sentence, and pronunciation
- **Trace canvas** — draw the letter with mouse or touch
- 🔊 Text-to-speech (native German/English voice at child-friendly speed)
- Tap to hear pronunciation, earn ⭐ stars

### 🔢 Numbers (1–10)
- Each number with matching objects and a fun sentence
- Tap to hear the number spoken aloud
- Visual object groupings for counting

### 🎨 Paint Studio
- Full canvas drawing (mouse + touch)
- 12 colour palette including white and black
- 4 brush sizes
- 🧹 Eraser · 🪣 Fill tool · 🗑️ Clear
- 16 sticker stamps (🌟☀️🌈🦋🎈 and more)
- 💾 Save painting as PNG

### 🎵 Nursery Rhymes (8 rhymes)
- Twinkle Twinkle · Baa Baa Black Sheep · Jack and Jill
- Humpty Dumpty · Incy Wincy Spider · Old MacDonald
- Rain Rain Go Away · ABC Song
- 🔊 Tap to hear full rhyme read aloud

### 🧠 Quiz
- Auto-generated questions from Letters + Numbers + Animals
- 4-option multiple choice
- +10 XP for correct answers
- Animated correct/wrong feedback
- Score tracking per session

### 🦁 Animals (30 animals)
- Dog, Cat, Lion, Tiger, Elephant, Penguin, Dolphin and 23 more
- Tap to hear: name + sound + fun fact
- Fun facts tooltip popup
